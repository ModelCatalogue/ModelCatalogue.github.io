Open Source

We are committed to the open-source community and the Model Catalogue is released under the MIT license. Use it, fork it, improve it!

Fork us on GitHub.

To get involved with the project feel free to create a feature or bug request on our GitHub issue tracker. Improvements and bugfixes are always welcome, just fork our project, make the changes on the forked repository, and then create a pull request.
